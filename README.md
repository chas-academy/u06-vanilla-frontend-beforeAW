
# U06 Vanilla Frontend

## Installation Instructions

1. Clone the repository:
   ```bash
   git clone <repo-url>
   cd u06-vanilla-frontend-beforeAW
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Build the project for production:
   ```bash
   npm run build
   ```
